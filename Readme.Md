A modular Telegram Python bot running on python3 with a sqlalchemy database and an entirely themed persona to make Aqua suitable for Anime and Manga group chats. 

Can be found on telegram as [Aqua](https://t.me/Aqua_Probot).

The Support group can be reached out to at [Support Aqua](https://t.me/Anime_Chat_XKaizuryu), where you can ask for help about [Aqua](https://t.me/xelcius), discover/request new features, report bugs, and stay in the loop whenever a new update is available.
